# HR Workflow Designer

This is a prototype HR workflow designer built using **React**, **Vite**, and **React Flow**.

## Features

- Drag-and-drop nodes (Start, Task, Approval, Automated Step, End)
- Connect nodes using arrows to build a workflow
- Right-side panel to edit node details (title, description, assignee, dates, actions)
- Test / Sandbox panel to run a simple simulation and view the execution log
- Import / Export workflow as JSON

## How to Run

```bash
npm install
npm run dev
